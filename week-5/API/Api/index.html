API

APIs are readymade set of code building blocks that allow a developer to implement programme that will otherwise be heard or impossible to implement

Browser API

Browser APIs are built into your wep browser and are able to expose data from the surrounding computer environment or do useful complex things for example:

  • The DON(document object model)

    The document object model api allows you to manipulate html and css by creating,removing and changing html dynamically apply new style to your page among other things.Everytime you see a popup window appear on your page or some new content display that is DOM in action like the example we did above

  • The Geo-Location Api

    The geo-location api retrieve geo-graphical information.This is how google maps is able to find your location and plot it on a map

  • The Canvas And WebGL Apis

    The canvas and webGL apis allows you to create animated 2D and 3D graphics

  • Audio And Video Apis Examples:Html Media Element

    Html media elements and webrtc allows you to do many interesting with media such as play audio and video and display it on someone else computer

Third Party APIs

Third party APIs are not built into the browser by default and you will generally have to grab their code and information from somewhere on the web,for example:

  • Bluesky API

    The bluesky APIs allows you to do things like displaying your lastest post on your website

  • Goole MapsAPI And Openstreetmap API

    Goole MapsAPI And Openstreetmap API allows you to embed custom map into your website and other such functionality
