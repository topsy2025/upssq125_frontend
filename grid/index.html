Css Grid Components

A grid always consists of:

  • A grid container which is the parent container
  • a grid items which is the items inside the container

Grid Container and Grid Items

A grid layout consists of a parent element(a grid container),with one or more grid items

All direct children of the grid container automatically become grid items


A grid layout must have a parent element with the displayproperties set to gridor inline-grid

Direct child element(s) of the grid container automatically become grid items
