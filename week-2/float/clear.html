when we use the float property and we want the next element to be bellow not to the right ot the left we will have to use the clear property

the clear propertyspecifieswhat to happen to the element that is next to the flowting element

The clear property can have one of the following value

  • None- The elementis not pushed bellow left or right floated elements and this is the defult
  • left- The element is pushed bellow left floated elements
  • right- The element is pushed bellow right floated elements
  • both- The element is pushed bellow both left and right floated elements
  • Inherit- The element Inherit the float propertyofit parent
