overflow property description

The overflow property specifies whether to click the content or to add stroll bars when the content of an element is too big to fit the specify area

overflow property has the following values visible,hidden,auto

Note:The overflow property only works for block element that has specify height.In second note,in oss lion is the name of an operative system on a mac computer and scroll bars are hidden by default and only show when been used even if the overflow is set on stroll

overflow-x and overflow-y

The overflow-x and overflow-y properties specifies whether to change the verflow content just horizontally or vertically or both

overflow-x specify what to do with the left/right edges of the content

overflow-y specify what to do with the top/bottom edges of the content



You can used the overflow property when you want to have better control of the layout.the overflow property specify what happens if content overflow an element box
