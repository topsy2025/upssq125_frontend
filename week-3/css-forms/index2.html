Html font

An html fonts is use to collect user input.The user input is most often sent to a server for pocessing.

Form Element

The html form element is used to create an html form for the user input:

The form element is a container for different types of input element such as text fills,check buttons,radio buttons and submit button

The Input Element

The html input element is the most used element

Type Description
Input type text It displays the single line text fill
Input type radio It defines a radio button input fill that allows selections of one option from a set
Check box Defines a check box page that allows selection of multiple option from a set
Input Type Number This define an input fill for entering a number
Input Type Email Defines an input fill for entering an email address
Input Type Password Defines a password input fill
Input Type File This allows the upload from a file from your computer
