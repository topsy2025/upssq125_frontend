Responsive web design(RWD)

Responsive web design(RWD) is a design approached that ensure a website looks and functions well across a wide range of devices and screen sizes. - From mobile phones to tablet to desktop computers - Without requiring separate version of site

Key concepts of responsive web design

  1. fluid grids:

    Layout are build using relative unit like percentage instead of fixed unit like pixels allowing element to resize based on the screen

  2. Flexible images

    Images scale within their containing elements so they overflow or distort on different screen sizes

  3. Fluid grids:

    Layout are built using relative unit like percentage instead of fixed unit like pixel allowing element to resize based on the screen

  4. Media query

    Css rules that apply style based on the device characteristics
