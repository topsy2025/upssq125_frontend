Css Flexbox

Flexbox is short for flexible box input model.

Flexbox is a layout method for arranging items in rows and columns.

Flexbox makes it easier to design a flexible responsive layout structure without using float or positioning.

Flex vs Grid

The css flexbox layoutshould be used for one dimensional layout in rows and columns.

The cssgrid layoutshould be used for two dimensional layout with rows and columns.

Css flexible box module

Before the flexible layout module,there were four layout modules:

  • Block for sections in the webpage.
  • Inline for text.
  • Table for two dimensional table data.
  • Position for explicit positioning of an element

Css flexbox is supported by every modern browsers

Css flexbox component

A flexbox also consists of:

  • A flex container
  • Flex items
