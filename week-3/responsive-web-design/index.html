Responsive web design(RWD)

Responsive web design(RWD) is a design approached that ensure a website looks and functions well across a wide range of devices and screen sizes. - From mobile phones to tablet to desktop computers -Without requiring separate version of site.

Key concepts of responsive web design:

  1. fluid grids:

    Layout are build using relative unit like percentage instead of fixed unit like pixels allowing element to resize based on the screen

  2. Flexible images:

    Images scale within their containing elements so they overflow or distort on different screen sizes

  3. Fluid grids:

    Layout are built using relative unit like percentage instead of fixed unit like pixel allowing element to resize based on the screen

  4. Media query

    Css rules that apply style based on the device characteristics such as screen width,height,resolution or orientation.

    Example:
    
                    @media (max-width: 768px) {
                        .menu{
                display: none;
            }
        }

    Mobile-first approach:

    Design start with the smallest screen size and scale up,ensuring a good user experience on mobile design first.

Gride options in responsive design

  • Css grid layout:

    Allows for complex layout with rules and column

  • Flexbox

    Provides a one dymational layout model that is great for alligning item in a row or color making it easier to create flexible layout.

  • Frame work:

    Framework like bulume,bootstrap,w3css,tailwind css and foundation provides prebuilt responsive grid systems that can be easily integrayed into project.
