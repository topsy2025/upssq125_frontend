Home

In css,we use the font-family property to specify the font of the text

Note:if the font name is more than one word,it must be in quotation marks,like:'Times new roman

Tips:The font-family property should hold several font name as a fall back system.The browser will try to use the first in the list,if it is not available it will try the others.

Example:

.p1{
font-family property:'Time New Roman',Times,serif; } .p2{
font-family:Ariel,Helvetica,sans-serif } .p3{
font-family:"Lucida console","Courier New",Monospace }
